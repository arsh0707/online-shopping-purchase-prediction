# online-shopping-purchase-prediction
Predicting whether an online shopper will make a purchase using Logistic Regression
## Online Shopper Purchase Prediction

This project uses a Logistic Regression model to predict whether an online shopper will make a purchase based on browsing behavior.

### Key Steps
- Data cleaning and preprocessing
- Data visualization
- Logistic Regression model training
- Model evaluation (88% accuracy)
- Prediction on new customer data

### Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
